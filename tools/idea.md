# IDEA

写些自己常用的`IDEA`快捷键.

首先,装上`IdeaVim`插件. 将快捷键认为`Default`.

`Ctrl + N`: 定位到类名所在的位置,需要在`IdeaVim`中修改
`Ctrl + Shift + N`: 定位到指定名称的文件
`Ctrl + /`: 注释当前行
`Ctrl + j/k`: 用于在列表中上下选择,相当于`Down/Up`
`Ctrl + E`: 选择最近打开文件
`Ctrl + J`: 插入模板
`Alt + 1`: 定位到`Run`底窗口
`Alt + 2`: 定位到`TODO`底窗口
`Alt + 3`: 定位到`Terminal`底窗口
`Alt + R`: 运行
`Alt + J`: 选择当前光标所在标识符,可继续选择,同时替换名称
`Alt + [/]`: 光标跳转到上/下方法定义外
